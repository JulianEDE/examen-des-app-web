# Examen Final - Desarrollo de Aplicaciones Web

## Descripción

Aplicación web desarrollada con Vue.js 3, Bootstrap 5 y MockAPI para la gestión de usuarios y productos mediante operaciones CRUD, autenticación y protección de rutas.

## Tecnologías Utilizadas

* Vue.js 3
* Vue Router
* Bootstrap 5
* Bootstrap Icons
* Axios
* MockAPI
* Git y GitHub

## Instalación

```bash
npm install
```

## Ejecución

```bash
npm run dev
```

## Estructura del Proyecto

* `/components` Componentes reutilizables
* `/views` Vistas principales
* `/router` Configuración de rutas
* `/services` Consumo de MockAPI
* `/assets` Recursos estáticos

## Funcionalidades

* Inicio de sesión
* Rutas protegidas
* CRUD de Usuarios
* CRUD de Productos
* Modales Bootstrap
* Alertas Bootstrap
* Diseño responsive
